# User manual — EasySave v2.0

## 1. Overview

EasySave v2.0 is a backup application with a WPF graphical interface.

It allows users to:

- create backup jobs;
- execute a selected job;
- execute several jobs sequentially;
- monitor execution progress;
- select the interface language;
- choose the log format;
- configure extensions to encrypt;
- block execution when a business software is running.
