# EasySave v2.0 — Technical support information

## 1. Target version

EasySave v2.0.0

## 2. Application type

EasySave v2.0 is a backup application developed in C# with .NET 10.0.

It includes:

- a WPF graphical interface;
- an MVVM architecture;
- a separated business core `EasySave.Core`;
- a logging library `EasyLog`;
- an external `CryptoSoft` integration.
